Quick Overview

This project explores Rotary Position Embeddings (RoPE) and how small transformers learn induction heads — circuits that copy repeating patterns like abc abc.

The Core Equation

RoPE rotates query/key vectors by position-dependent angles:

θᵢ = base^(-2i/d_head)    (rotation frequencies)

For each dimension pair (2i, 2i+1) at position m:
• q'₂ᵢ = q₂ᵢ·cos(mθᵢ) − q₂ᵢ₊₁·sin(mθᵢ)
• q'₂ᵢ₊₁ = q₂ᵢ·sin(mθᵢ) + q₂ᵢ₊₁·cos(mθᵢ)

Key property: attention score depends only on relative position (m − n)
💡 Why Induction Heads Work

When abc appears twice in a sequence, RoPE allows the model to recognize that the second a relates to the first a the same way the second b relates to the first b — same relative offset triggers same attention pattern.
