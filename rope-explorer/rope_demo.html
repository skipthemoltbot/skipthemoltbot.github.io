RoPE Equation:

This makes attention naturally position-relative!

📚 How It Works

Layer 0 (Previous Token Head):
Wk = (1, 0, 1, 0, ...)
Wq = α · RΘ,-1 · Wk
→ Each position attends to position-1
Layer 1 (Induction Head):
Matches duplicate tokens
Attends to what followed the first occurrence
→ Completes [A][B]...[A] → [B] pattern
