3. Attention Score Derivation

Step 1: Query and Key Computation

For token embedding $\mathbf{x}$ at position $m$:

$$\mathbf{q}_{\text{raw}} = \mathbf{x} W_q \in \mathbb{R}^{d_{\text{head}}}$$ $$\mathbf{k}_{\text{raw}} = \mathbf{x} W_k \in \mathbb{R}^{d_{\text{head}}}$$
Step 2: RoPE Rotation

For each dimension pair $(2i, 2i+1)$ at position $m$:

$$q'_{2i} = q_{2i} \cos(m\theta_i) - q_{2i+1} \sin(m\theta_i)$$ $$q'_{2i+1} = q_{2i} \sin(m\theta_i) + q_{2i+1} \cos(m\theta_i)$$
Step 3: Dot Product with RoPE
$$\mathbf{q}'^T \mathbf{k}' = \sum_i \left[q'_{2i}k'_{2i} + q'_{2i+1}k'_{2i+1}\right]$$

Expanding and using trig identities:

$$= \sum_i \left[\cos((m-n)\theta_i)(q_{2i}k_{2i} + q_{2i+1}k_{2i+1}) + \sin((m-n)\theta_i)(q_{2i}k_{2i+1} - q_{2i+1}k_{2i})\right]$$

✓ Critical Result

The dot product depends ONLY on $(m - n)$ — the relative position! This is the fundamental property that enables previous token heads.

4. Previous Token Head: Derivation

Goal: Maximize score at offset $\Delta = 1$

For offset $\Delta = 1$:
$$\text{score}(m, m-1) = \sum_i \left[\cos(\theta_i)(q_{2i}k_{2i} + q_{2i+1}k_{2i+1}) + \sin(\theta_i)(q_{2i}k_{2i+1} - q_{2i+1}k_{2i})\right]$$

Using $i=0$ values: $\cos(\theta_0) = 0.5403$, $\sin(\theta_0) = 0.8415$

6. Proof: Why Both Approaches Work

Functional Equivalence

Both hand-coded and learned weights satisfy:

$$\mathbb{E}[\text{score}(m, m-1)] \gg \mathbb{E}[\text{score}(m, n)] \text{ for } n \neq m-1$$

Hand-coded: $W[i,j] \in \{-1, +1\}$ — perfect orthogonal basis

Learned: $W[i,j] \in \mathbb{R}$ — distributed weights that statistically align with RoPE phases

✓ Conclusion

SGD finds a solution in a different subspace of the weight space that achieves functional equivalence. The hand-coded alternating pattern is one solution; SGD discovers another that's easier to reach via gradient descent and more robust to perturbations.
